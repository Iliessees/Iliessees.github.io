Aperçu du Projet

SAE S2.03 - Administration Système

Ce projet s'inscrit dans le cadre de ma formation en BUT Informatique à l'IUT2 de Grenoble. Il consiste en la création d'un guide d'installation complet pour mettre en place un environnement de développement web sur une machine virtuelle Linux.

L'objectif principal était de maîtriser l'installation et la configuration de services essentiels pour le développement web, en partant d'une installation minimal de Debian 12 jusqu'à un serveur web fonctionnel avec base de données.

Détails du Projet

Guide d'Installation : Serveur Web Complet

Description

Création d'un guide d'installation détaillé pour configurer un environnement de développement web complet sur machine virtuelle. Le projet couvre l'installation de Debian 12 sans interface graphique, la configuration d'Apache comme serveur web, l'installation de PostgreSQL pour la gestion de base de données, PHP pour le développement côté serveur, et PHPpgAdmin pour l'administration graphique des bases de données.

Objectifs

  • Maîtriser l'installation de Debian 12 en mode minimal
  • Configurer et sécuriser un serveur Apache
  • Installer et configurer PostgreSQL
  • Intégrer PHP avec Apache pour le développement web
  • Documenter le processus pour faciliter la reproduction
